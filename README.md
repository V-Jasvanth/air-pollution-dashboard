# 🌍 IoT Air Pollution Monitoring Dashboard

A web-based dashboard for monitoring air pollution parameters using IoT technology.  
This project simulates environmental sensor data and visualizes it through an interactive dashboard.

## 🚀 Live Website
👉 https://air-pollution6872.netlify.app/

## 📊 Features
- Real-time AQI monitoring
- Temperature and Humidity display
- Dynamic color alerts
- 2-second auto refresh updates
- AQI history graph visualization
- Dark / Light mode toggle
- Admin login panel with password protection
- Adjustable threshold settings

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Chart.js
- Netlify (deployment)
- GitHub (version control)

## 🔧 Hardware Components (for IoT integration)
- ESP32 Development Board
- MQ135 Air Quality Sensor
- DHT11 Temperature & Humidity Sensor
- Breadboard
- Jumper Wires
- Power Supply

## 🧠 System Architecture
Sensors → ESP32 → WiFi → Web Dashboard → Admin Control Panel
The ESP32 collects environmental data from sensors and sends it to the dashboard for real-time monitoring and visualization.
## 🔐 Admin Access
The dashboard includes a secure admin panel where administrators can:
- Configure AQI thresholds
- Enable or disable alerts
- Monitor system parameters

## 📱 Access the Dashboard
The system can be accessed through any web browser on:
- Laptop
- Smartphone
- Tablet

## 🎓 Project Purpose
This project demonstrates how IoT technology can be used to monitor environmental pollution levels in real time and provide actionable insights through a web-based dashboard.
